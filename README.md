# BackbarCodingChallenge
A list app code challenge for Backbar

### Prequisites
`npm 6.2.x`
`heroku 7.27.x`

### Installing
Run `npm install`.

### Building
Run `npm run build`.

### Running
Run `npm start`.

### Deploying to Heroku
Create a new app, run `heroku create`
Deploy app to Heroku, run `git push heroku master`
